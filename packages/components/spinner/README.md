# @banyu/loading

Spinner express an unspecified wait time or display the length of a process.

Please refer to the [documentation](https://localhost:3000/docs/components/loading) for more information.

## Installation

```sh
yarn add @banyu/loading
# or
npm i @banyu/loading
```
