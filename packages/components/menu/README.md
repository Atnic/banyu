# @banyu/menu

A menu displays a list of options and allows a user to select one or more of them.

Please refer to the [documentation](https://localhost:3000/docs/components/menu) for more information.

## Installation

```sh
yarn add @banyu/menu
# or
npm i @banyu/menu
```
