# @banyu/dropdown

Displays a list of actions or options that a user can choose.

Please refer to the [documentation](https://localhost:3000/components/dropdown) for more information.

## Installation

```sh
yarn add @banyu/dropdown
# or
npm i @banyu/dropdown
```
