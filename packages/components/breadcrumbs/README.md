# @banyu/breadcrumbs

Breadcrumbs display a hierarchy of links to the current page or resource in an application.

Please refer to the [documentation](#) for more information.

## Installation

```sh
yarn add @banyu/breadcrumbs
# or
npm i @banyu/breadcrumbs
```
