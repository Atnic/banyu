# @banyu/system

Banyu system primitives, here you can find the `extendVariants` utility, `BanyuProvider` and some other utilities.

Please refer to the [documentation](https://Banyu.org) for more information.

## Installation

```sh
yarn add @banyu/system
# or
npm i @banyu/system
```
